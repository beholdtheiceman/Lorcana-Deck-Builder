# Lorcana Tech Cards Reference
_Last updated: July 2026 (Sets 1–6 / Azurite Sea era)_

"Tech cards" are situational includes added to a deck specifically to address meta threats. When a user asks "what should I run against X" or "what's good tech for Y meta," reference this file.

## Removal Tech
_Use when specific threats need to be answered_

| Card | Ink | Cost | Effect |
|------|-----|------|--------|
| Fire the Cannons! | Ruby | 1 | Deal 2 damage to chosen character |
| Smash | Ruby | 3 | Deal 3 damage to chosen character |
| Dragon Fire | Ruby | 5 | Deal 5 damage to chosen character |
| Be Prepared | Amethyst | 7 | Banish all characters with 2 strength or less |
| Burn That Cookbook! | Ruby | 2 | Banish a chosen item |
| Return to the Sea | Amethyst | 3 | Return chosen character to owner's hand |
| Hypnotize | Amethyst | 3 | Chosen character can't quest or challenge until your next turn |

**Removal Selection Guide:**
- Vs. low-willpower aggro → Fire the Cannons!, Be Prepared
- Vs. high-willpower threats → Dragon Fire, challenging with a Challenger character
- Vs. key items → Burn That Cookbook!
- Vs. Ward characters → You CANNOT use targeted removal; must challenge directly or use board wipes
- Vs. Evasive characters → Can only challenge with Evasive characters; removal spells work fine

## Situational Tech by Matchup

### vs. Ruby/Sapphire Midrange
- **Run:** Ward characters (they run mostly targeted removal), Bounce effects for their key threats, Be Prepared if their board floods
- **Avoid:** Slow cards that don't affect the board before turn 5

### vs. Amethyst/Steel Control
- **Run:** Ward characters, rush threats that apply early pressure before control stabilizes, redundant threats so bounce never really removes your plan
- **Avoid:** Single large threats that get bounced repeatedly; expensive cards that play into their gameplan

### vs. Emerald/Ruby Aggro
- **Run:** Bodyguard characters, cheap removal (Fire the Cannons!, Smash), Be Prepared for the board flood, healing effects
- **Avoid:** 5+ cost cards in your opening curve; you need to stabilize before turn 4

### vs. Amber/Sapphire Ramp
- **Run:** Aggressive pressure to race before their big threats land; removal for their ramp pieces; Rush characters
- **Avoid:** Control strategies that let them reach 7+ ink; you need to end the game fast

## How to Use This File

When making tech recommendations:
1. Identify the primary meta threat the user wants to answer
2. Check the relevant section above for candidates
3. Filter by the user's existing ink colors (tech must fit the deck's color identity)
4. Recommend 2–4 copies of the best fit, explaining the interaction
5. Note which slots to cut from the existing list (usually the weakest 4-of in the matchup)
