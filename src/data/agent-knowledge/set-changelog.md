# Lorcana Set Changelog — Meta Impact by Set
_Reference for understanding how each set shifted the competitive landscape_

Use this when a user asks about a specific set, wants historical context, or asks "why is [card] good now?"

---

## Set 12 — Wilds Unknown (Current Set)

**Meta Impact: Major — Established new dominant archetypes**

### Key Cards Added
- **Woody (1-cost):** Best 1-drop in current meta. Shift line into Woody (5) is a premier turn-3 play.
- **Sid Phillips (6-cost):** Forces opponent to banish their own character. Does NOT specify your toys — works in any toy mirror.
- **Headless Horseman (5-cost):** Catch-all removal. 4-of in Amber Ruby Toys.
- **Jesse Woodard:** Card draw + shrinks opposing challengers. Non-once-per-turn triggers.
- **Babyhead:** Draw engine + combat power buff when paying 2 or less. Enables small toys to trade up.
- **Bullseye:** Sings You've Got a Friend in Me — finds 2 toys from top 4. Honorary 2-drop.
- **Bouncing Ducky:** Costs 1 less per toy in discard — often free; 5/5/2 body.
- **Candy Drift:** Action — lets small characters trade into ANY large character.
- **You've Got a Friend in Me:** Song — look at top 4, add 2 toys to hand. One of the most powerful archetypal songs ever printed.
- **Akood:** Ramp enabler for Amber Steel.
- **Merryweather:** 4/5 Ward — Blurple's new premier blocker (replacing Genie in many builds).
- **Enigmatic Inkcaster:** Item — creates ongoing burn lore effect; no item removal in Amber/Ruby.
- **Snowfort:** Location — ongoing burn effect; similarly unchecked by many decks.
- **Dash Parr:** Burn lore character — generates lore without questing.
- **Giant Cobra:** Burn lore effect.

### Format Shifts in Set 12
- **Amber/Ruby Toys emerges** as a competitive Tier 2 deck — 51/60 cards from Set 12
- **Toys theme** becomes cross-color (Amber/Ruby and Amber/Emerald variants)
- **Burn lore** becomes a viable alternate win condition — bypasses board state entirely
- **Dogs (Amber/Emerald)** consolidates as Tier 1 with Under the Sea synergies
- **Blurple (Amethyst/Sapphire)** evolves — Merryweather replaces Genie in many lists

### Cards That Aged Well into Set 12
- Demona (Blurple core)
- Elsa shifted (Blurple/YP Dale)
- Dale (YP Dale shell)
- Cheshire Cat (Blurple/YP Dale/Amethyst Steel disruption)
- Yzma (Amethyst Steel card advantage engine)
- Goliath (Sapphire Steel draw engine)

### Rotation Awareness
- Most competitive Set 12 decks are 80–90% Sets 9–12 cards
- Amber Ruby Toys is 51/60 cards from Set 12 alone — extremely rotation-proof
- Dogs and Blurple also heavily Set 9+ — expect minimal impact from rotation

## Set 1 — The First Chapter
**Release:** August 2023  
**Size:** 204 cards

**Meta impact:** Established the core archetypes. Ruby/Sapphire emerged immediately as the most consistent pairing. Amethyst/Steel control was viable but slower. Emerald aggro was playable.

**Key additions to the game:**
- All 6 ink colors established
- Core keywords introduced: Shift, Rush, Evasive, Ward, Bodyguard, Challenger, Support, Reckless, Resist
- Foundational staples that remain relevant: Fire the Cannons!, Be Prepared, Dragon Fire, Let It Go, A Whole New World
- **Stitch - Carefree Surfer** (Ruby): defined early Ruby aggression
- **Elsa - Snow Queen** (Sapphire): the draw + body template
- **Maleficent - Monstrous Dragon** (Amethyst): the first massive board-wipe threat
- **Moana - Of Motunui** (Sapphire): established the efficient low-cost quester template

## Set 2 — Rise of the Floodborn
**Release:** November 2023  
**Size:** 204 cards

**Meta impact:** Introduced Shift in a major way. Amethyst/Steel gained significant tools. Mid-range strategies became more defined.

**Key new mechanics:** Shift became a primary build-around; many powerful Shift targets printed.

**Key additions:**
- **Shift cards** across all colors made decks more flexible and consistent
- **Genie - On the Job** (Amethyst): 2-for-1 draw effect redefined Amethyst consistency
- **Merlin - Rabbit** (Sapphire): the first efficient ramp character
- **Beast - Relentless** (Steel): massive Ward threat that resists most removal
- **Cinderella - Stouthearted** (Sapphire): cheap Shift target, redefined Sapphire curve options
- Songs became more prominent as high-value plays
- Ruby gained more targeted removal options

**Format shift:** Amethyst/Steel rose from fringe to Tier 2 on the strength of new Shift targets and bounce support.

## Set 3 — Into the Inklands
**Release:** February 2024  
**Size:** 204 cards

**Meta impact:** Locations introduced as a new card type. Ramp strategies became more viable. Aggro got faster tools.

**Key new mechanics:** **Locations** — a permanent card type that characters can move to, generating effects while there. Changed board evaluation significantly.

**Key additions:**
- **Locations** across all colors added a new axis of play (contested territory, lore from locations)
- **Peter Pan - Never Landing** (Emerald): cemented Emerald/Ruby aggro as a real archetype
- **Tamatoa - So Shiny!** (Sapphire): enormous ramp payoff card
- **Fishbone Quill** (Sapphire): the best ramp item in the game
- **Tiana - Celebrating Mardi Gras** (Amber): powerful Support character
- Locations added a new resource to contest — decks needed to decide whether to play location-based or traditional strategies

**Format shift:** Emerald/Ruby aggro rose to Tier 1. Amber/Sapphire ramp became a real strategy with Fishbone Quill + Tamatoa.

## Set 4 — Ursula's Return
**Release:** May 2024  
**Size:** 204 cards + 12 Special Rarities

**Meta impact:** Major power creep; several of the strongest individual cards in the game printed. The meta became more complex.

**Key new mechanics:** **Special Rarities** (foil enchanted versions with unique art); expanded Songs.

**Key additions:**
- **Ursula - Deceiver of All** (Amethyst): direct lore drain, completely novel win condition
- **Maui - Hero to All** (Sapphire): strong all-around card that anchored Ruby/Sapphire
- **Sisu - Empowered Sibling** (Sapphire): card that boosted lore output significantly
- **Maleficent - Biding Her Time** (Amethyst): Ward + draw on a 4-drop, immediately Tier 1
- **A Pirate's Life** (Emerald): cheap bounce gave Emerald control options
- Power level of individual cards increased significantly — deckbuilding margins tightened

**Format shift:** Ruby/Sapphire solidified as Tier 1. Amethyst gained several strong new tools. The meta stabilized around the Tier 1-2 breakdown described in meta-archetypes.md.

## Set 5 — Shimmering Skies
**Release:** August 2024  
**Size:** 204 cards

**Meta impact:** Flight (Evasive) themes expanded. New ink color combinations became more viable. Control tools improved.

**Key additions:**
- Expanded Evasive character support across multiple colors
- New multi-lore questing characters challenged the efficiency standard
- **Baymax - Healthcare Companion** (Steel): new Resist + Bodyguard option
- Additional healing and support tools for Amber
- More efficient Ward characters printed
- Ruby gained new Rush options at lower costs

**Format shift:** Evasive-heavy strategies saw a resurgence. Ward characters became more important tech choices.

---

## Set 6 — Azurite Sea
**Release:** November 2024  
**Size:** 204 cards

**Meta impact:** Most recent set as of this knowledge base. Sea/nautical theme. Expanded multi-character synergies.

**Key new mechanics:** Expanded "Sing Together" — multiple characters can contribute to singing a high-cost song together.

**Key additions:**
- **Sing Together** made high-cost songs accessible in aggressive and midrange decks
- New Challenger characters across Ruby and Emerald refined aggressive strategies
- Sapphire received additional card draw options at the 2–3 cost range
- Steel got new large Resist characters making the color's late game stronger
- Amber received better mid-cost Support characters

**Format shift:** The meta at Set 6 reflects the Tier 1-2 breakdown in meta-archetypes.md. Ruby/Sapphire remains the most consistent, Emerald/Ruby aggro is the fastest, Amethyst/Steel is the control option.

## Notes for Format Awareness

- **Standard Format:** Only the most recent sets are legal. As of Set 6, typically Sets 3–6 + current season product.
- **All-Sets Format (Illumineer's Challenge / Open):** Every set is legal. Much wider card pool; early set staples like Fire the Cannons! (Set 1) remain relevant.
- Always ask the user which format they're building for — card availability changes dramatically.

## Cards to Know That Have Aged Poorly

These were once strong but are now often outclassed:
- Early Shift targets from Set 1 without Ward — now often replaced by Set 3-4 versions
- Low-willpower Bodyguard characters (pre-Set 3) — higher stats printed later
- Single-ink draw cards at 5+ cost without added effects — outclassed by newer draw options

## Cards That Have Aged Well (Format Staples Across All Sets)

These remain relevant in almost any deck that runs their ink color:
- **Fire the Cannons!** (Ruby, Set 1) — still the best 1-cost removal
- **Be Prepared** (Amethyst, Set 1) — still the best board wipe
- **Let It Go** (Sapphire, Set 1) — still the best draw song
- **Fishbone Quill** (Sapphire, Set 3) — best ramp item
- **Dragon Fire** (Ruby, Set 1) — reliable 5-damage removal
